Misc commands
Command Description Example usage
!info Provides current information on the bot. !info
!sound Makes the bot join your current voice channel and play the specified sound.
!sound list to list all the available sounds !sound stop to stop playing a sound		 !sound ggm8
!steam Provides some information about Steam servers status. !steam
!tracktwitch Makes the bot post a message in the general channel when the specified user starts streaming. !tracktwitch @Minipasglop#3347
